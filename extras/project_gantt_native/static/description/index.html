From Numbers Data to Visual Representation With Odoo Gantt View
What are the most important parameters of any work or task assigned to your employee or team member:
Deadline
Task Progress rate
And current Status of task
But how will you track the tasks when there are hundreds of them at the same time?
We present the Gantt View For the Odoo Project Menu.
What does it Do?
  • Odoo Project Gantt View converts these parameters from numbers to visual representation on the Chart.
  • Each task can be easily identified as horizontal with a color.
  • The length of the bar directly shows the deadline of that task.
  • Each task can be sorted out as per the employee, current status, time, project, etc
How to use the Odoo Gantt View?
Use Task Filters
Sort out the desired tasks by the number of filters provided in the View.
  • Sort the tasks as per the current stage - new, in progress, done, etc.
  • Sort by employee or assigned to.
  • Group tasks according to the project they belong to.
  • Visually see the number of tasks at each employee's end.
  • See the current tasks in each project and their progress all in one screen.
Use time Period Filter to track task progress
  • Assign different time filters to track the progress of the tasks.
  • TIme periods available - hours, Day, week, month, year
  • The length of the colored horizontal task bar show the deadline of the task as per the filter selected.
  • You can Visually see which tasks and projects are exceeding deadlines.
  • Make smarter decisions to improve efficiency.

Access the Gantt View In the Project >> All tasks Menu

Use Filters To Track And Manage Tasks via Odoo Gantt View

  • Sort Tasks according to the current status or stage such as work in progress, Done, pending, etc
  • Sort the tasks according to individual projects. All the task in one project are grouped together.
