Mail Messages Draft - Pro Version

All features of basic version plus:

  • Compose new message directly from "Messages" menu
  • Restore message from draft in composer. Select which draft version to restore
  • Add labels to drafts to identify different versions easier
  • Move draft to other record
  • Add followers to draft to collaborate on draft
